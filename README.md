# go-microservices
Learning to build quick and easy microservices with Go

### How to Run 

Step 1 : Install docker desktop from [here](https://www.docker.com/products/docker-desktop/)

Step 2 : Go to projects folder and run 

```make up_build```

Step 3: run frontend app 

`make start`

Step 4: go to [localhost](http://localhost/) for testing template
